# pyhangouts  

A selenium-based client for Google Hangouts

----
## Environments
- Ubuntu 14.04  
- Python 3.4.3

## Usage
    >>> from pyhangouts.client import Firefox
    >>> ff = Firefox()
    Account: `Enter your Google account`
    Password: `Enter your password (won't reveal on screen) `
    >>> ff.chat('This is a test message')
